# Car Sales System – C++ Project

## Përshkrimi
Ky projekt është një sistem i thjeshtë për shitjen e makinave, i zhvilluar në gjuhën programuese **C++**, duke përdorur konceptet bazë të **Object-Oriented Programming (OOP)**, kontrollin e rrjedhës së programit dhe menaxhimin e input-it të përdoruesit.
